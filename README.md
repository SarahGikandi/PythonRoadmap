# Python Focus Tracker

A minimal React application designed to help me stay disciplined while learning Python step-by-step without rushing ahead. 

The app focuses on:
- structured weekly learning
- daily progress tracking
- consistency
- reflection
- reducing the habit of jumping ahead too quickly

## ✨ Features

- 4-week Python learning roadmap
- Daily task tracking
- Confidence rating system
- Daily mini logs
- Anti-jump mindset panel
- Focus Mode ("One Day Only")
- Responsive design (phone + laptop)
- Firebase-ready architecture
- Cross-device sync support (planned with Firebase) to avoid having different data on the same app on different devices.


## 🧠 Purpose

This project was built to solve a common learning problem:

> getting overexcited, skipping foundations, and losing consistency.

Instead of focusing on speed, the app encourages:
- steady progress
- one topic at a time
- discipline over pressure


## 🛠 Tech Stack

- CSS / Tailwind CSS
- Firebase (planned integration)

## ⏳ Why Only 4 Weeks?

The 4-week structure may seem short at first, but this roadmap is not designed for learning programming from zero.

It is a focused restart and restructuring journey.

I already have prior experience with:
- Python basics
- Java
- JavaScript
- SQL and databases
- OOP concepts

Because of this, many topics are revision-based rather than completely new.

The goal is not to rush through Python, but to:
- rebuild consistency
- strengthen foundations
- organize knowledge properly
- avoid the habit of jumping ahead too quickly

This roadmap focuses more on discipline, fluency, and structure than speed.
